## 🔧 核心功能

| 功能 | 命令 | 说明 |
|------|------|------|
| 添加 | `add <描述>` | 添加新待办事项 |
| 删除 | `delete <编号>` | 删除指定编号的待办 |
| 完成 | `complete <编号>` | 标记为完成（带时间戳） |
| 列表 | `list [--pending]` | 列出所有/未完成待办 |
| 统计 | `stats` | 显示完成率等统计 |
| 清空 | `clear` | 清空所有待办 |

## 💾 存储格式

**位置**: `/root/.nanobot/workspace/todos/TODO.json`

```json
[
  {
    "task": "学习 Rust 编程语言",
    "status": "completed",
    "created_at": "2026-03-04T07:43:23.904249",
    "completed_at": "2026-03-04T07:43:24.000803"
  },
  {
    "task": "阅读《道德经》",
    "status": "pending",
    "created_at": "2026-03-04T07:43:23.957380",
    "completed_at": null
  }
]
```

**数据字段**:
- `task`: 任务描述
- `status`: pending/completed
- `created_at`: ISO 格式创建时间
- `completed_at`: ISO 格式完成时间（可选）

## 💡 设计特点

1. **简单直观**: 命令行接口，易于使用

5. **统计功能**: 完成率计算
