### 📋《秘旨》主要内容（卷上 + 卷下）

#### 卷上 - 基础炼丹法
| 章节 | 内容 |
|------|------|
| **药物** | 铅、硫、汞的配比与处理（铅 10 两、硫 2 两） |
| **神室法象** | 金合子、水筦子、白金水海的制作方法 |
| **外鼐** | 瓷器鼎器的规格与安装 |
| **运水火符候** | 子午卯酉火候调控，半月开鼎看丹色 |
| **再水火断魂法** | 花银淬炼、三黄煮水、金液还丹制作 |
| **词曲** | 《满庭芳》《沁园春》丹道诗词 |

#### 卷下 - 进阶炼丹法
| 章节 | 内容 |
|------|------|
| **铅汞归祖既济图** | 铅汞提炼与归元方法 |
| **断魂之法** | 丹种沐浴、煅炼、真铅制作 |
| **还丹九转** | 第一转金砂黄芽、第二转混元神朴、第三转通天彻地 |
| **鼎器图** | 多种丹鼎设计图（原文有图但文本丢失） |

---

### 📋《秘旨传》主要内容

#### 传承故事
| 情节 | 内容 |
|------|------|
| **作者** | 西蜀孟君三世孙 孟煦 |
| **时间** | 南宋嘉定戊寅 (1218) → 宝庆乙酉 (1225) |
| **地点** | 福建三山 → 白鹤洞天 → 峨嵋西峰 |
| **师承** | 彭鹤林 → 兰元白（白玉蟾化身）→ 孟煦 |

#### 关键发现
> "此书玉蟾子为彭君内学不明，以平叔外丹之旨印可，仿言真铅一二，是书于探铅结胎分明，法象并火符缺欠斯明"

**意思是**：白玉蟾给彭耜的版本有所保留，内丹部分不清楚，外丹部分参考了张伯端（平叔）的理论。

#### 传承正统性
- 源自**轩辕黄帝 → 广成子**（崆峒山传授）
- 汉代：天师魏伯阳（《周易参同契》作者）
- 唐代：叶靖能、马明生、杜光明
- 晋代：吴许二君、葛金果
- 宋代：白玉蟾 → 彭耜 → 兰元白 → 孟煦
